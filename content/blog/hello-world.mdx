## 写在前面

终于开始了！折腾了很久，今天终于把个人博客网站搭起来了。

## 为什么写博客

写作是最好的学习方式。把学到的东西整理成文字，既能加深理解，也能帮助他人。

## 技术选型

这次选择了 **Next.js 15** 作为框架，搭配 **Tailwind CSS** 做样式，**MDX** 来写文章。

### 选择理由

- Next.js 的 App Router 提供了更好的页面组织方式
- Tailwind CSS 让样式开发非常高效
- MDX 可以在 Markdown 中使用 React 组件

## 半液态设计风格

UI 设计采用了半液态玻璃态风格：

- 深色渐变背景
- 毛玻璃效果的卡片
- 流体动画的光晕背景
- 渐变色文字

这种风格既有科技感，又不失优雅。
