# 个人博客网站

> 基于 Next.js 15 + Tailwind CSS + MDX 构建的个人博客、日记与开发记录网站

- **项目编号**: 012
- **创建日期**: 2026-06-22
- **技术栈**: Next.js 15, TypeScript, Tailwind CSS, MDX, Framer Motion
- **状态**: 已完成

## 项目目标

打造一个具有半液态玻璃态设计风格的个人博客网站，支持博客文章、日记记录和开发日志三种内容类型。

## 目录结构

| 目录 | 说明 |
|------|------|
| src/app/ | Next.js App Router 页面 |
| src/components/ | 可复用 UI 组件 |
| src/lib/ | 工具函数和数据读取 |
| content/ | MDX 内容源文件 |
| docs/ | 项目文档 |

## 使用方法

\\\ash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 启动生产服务器
pnpm start
